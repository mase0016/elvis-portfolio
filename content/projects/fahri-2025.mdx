## Project Overview

This project is a personal portfolio website designed to showcase various projects, skills, and experiences. It serves as a central hub for potential clients or employers to learn more about my work.

## Features

- **Dynamic Project Pages**: Each project has its own dedicated page, loaded from MDX files.
- **Responsive Design**: Optimized for various screen sizes, from mobile to desktop.
- **Contact Form**: A functional contact form for inquiries.
- **Smooth Animations**: Utilizes GSAP for subtle and engaging animations.

## Technologies Used

- **Next.js**: For server-side rendering, routing, and API routes.
- **React**: For building interactive user interfaces.
- **TypeScript**: For type safety and improved code quality.
- **Tailwind CSS**: For rapid UI development and styling.
- **MDX**: For writing content with JSX components.
